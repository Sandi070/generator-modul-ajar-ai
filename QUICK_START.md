## 📝 Cara Menggunakan:

### 1. Buka Aplikasi
- Buka file `index.html` di browser
- Atau gunakan local server: `python -m http.server 8000`

### 2. Test Aplikasi
- Buka `test-working.html` untuk memverifikasi semua fungsi bekerja
- Harus melihat semua test PASSED

### 3. Isi Form
1. Isi semua field yang wajib:
   - Nama Penyusun
   - Instansi
   - Mata Pelajaran (pilih dari dropdown)
   - Fase/Kelas (contoh: E/X)
   - Semester (pilih Ganjil/Genap)
   - Tahun Pelajaran (contoh: 2025/2026)
   - Materi/Topik
   - Alokasi Waktu (angka)
   - Model Pembelajaran (pilih dari dropdown)
   - Capaian Pembelajaran
   - Tujuan Pembelajaran

2. Pilih pendekatan pembelajaran (opsional):
   - Pendidikan Karakter
   - Moderasi Beragama
   - Bilingual
   - Berbasis TIK
   - Pembelajaran Inovatif

3. Klik "Generate Modul Ajar dengan AI"

4. Tunggu beberapa detik

5. Hasil akan muncul di bawah form

### 4. Lihat Hasil
- Scroll ke bawah untuk melihat modul yang dihasilkan
- Modul lengkap dengan:
  - Informasi Umum
  - Langkah Pembelajaran (sesuai model yang dipilih)
  - Asesmen
  - Pengayaan & Remedial

## 🐛 Jika Masih Ada Masalah:

### Test 1: Buka Browser Console
1. Tekan F12
2. Pilih tab "Console"
3. Cari pesan error berwarna merah
4. Screenshot dan laporkan

### Test 2: Jalankan Test File
1. Buka `test-working.html`
2. Lihat hasil test
3. Semua harus hijau (success)
4. Jika ada merah (error), screenshot

### Test 3: Cek Koneksi Internet
- Aplikasi membutuhkan internet untuk CDN libraries
- Jika offline, beberapa fitur mungkin tidak bekerja
