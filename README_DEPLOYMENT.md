# 🌐 Generator Modul Ajar AI - Versi Online

Aplikasi Generator Modul Ajar AI berbasis web yang dapat diakses secara online.

## 🚀 Status Deployment

Aplikasi ini siap untuk di-deploy ke platform hosting gratis seperti:
- ✅ Netlify (Direkomendasikan)
- ✅ Vercel
- ✅ GitHub Pages
- ✅ Render

## 📦 Fitur Aplikasi

- ✅ Generate modul ajar otomatis dengan AI
- ✅ Sesuai Kurikulum Berbasis Cinta Kemenag RI
- ✅ Export ke Word, PDF, dan HTML
- ✅ Responsive design (mobile-friendly)
- ✅ Offline-capable (PWA)
- ✅ Tidak perlu database atau backend

## 🎯 Cara Deploy

### Opsi 1: Deploy Cepat (5 Menit)
Lihat: `QUICK_DEPLOY.md`

### Opsi 2: Panduan Lengkap
Lihat: `DEPLOYMENT_GUIDE.md`

## 🔧 Teknologi

- HTML5, CSS3, JavaScript (Vanilla)
- Libraries: docx.js, FileSaver.js, jsPDF
- No backend required (100% client-side)
- PWA-ready dengan Service Worker

## 📱 Browser Support

- ✅ Chrome/Edge (Recommended)
- ✅ Firefox
- ✅ Safari
- ✅ Mobile browsers

## 🎓 Penggunaan

1. Buka aplikasi di browser
2. Isi form identitas modul
3. Pilih pendekatan dan model pembelajaran
4. Klik "Generate Modul Ajar dengan AI"
5. Download hasil dalam format Word/PDF/HTML

## 🔒 Keamanan & Privacy

- ✅ Semua proses di client-side (browser)
- ✅ Tidak ada data yang dikirim ke server
- ✅ Privacy-friendly
- ✅ HTTPS enabled (saat di-deploy)
