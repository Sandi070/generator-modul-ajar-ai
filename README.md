# 🎓 Generator Modul Ajar AI - Kurikulum Berbasis Cinta

Aplikasi web untuk menghasilkan modul ajar otomatis menggunakan AI sesuai dengan format Kurikulum Berbasis Cinta yang dicanangkan oleh Dirjen Pendis Kementerian Agama RI.

## ✨ Fitur Utama

### 📚 Mata Pelajaran Lengkap
- **Mata Pelajaran Umum**: Matematika, Fisika, Kimia, Biologi, Bahasa Indonesia, Bahasa Inggris, Sejarah, Geografi, Ekonomi, Sosiologi
- **Mata Pelajaran Agama**: Al-Quran Hadits, Akidah Akhlak, Fiqih, Sejarah Kebudayaan Islam, Bahasa Arab

### 🎯 Pendekatan Pembelajaran Terintegrasi
- **Pendidikan Karakter**: Mengintegrasikan nilai-nilai karakter mulia
- **Moderasi Beragama**: Menanamkan sikap toleransi dan moderasi
- **Bilingual**: 
  - Indonesia-Inggris (untuk mata pelajaran umum)
  - Indonesia-Arab (untuk mata pelajaran agama)
- **Berbasis TIK**: Memanfaatkan teknologi informasi dan komunikasi
- **Pembelajaran Inovatif**: Metode pembelajaran kreatif dan modern

### 🤖 AI-Powered Generation
- Generasi otomatis konten modul ajar
- Penyesuaian berdasarkan mata pelajaran dan pendekatan
- Struktur sesuai format standar Kementerian Agama

### 📥 Export ke Microsoft Word
- Download dalam format .docx yang rapi dan profesional
- Format siap cetak dan siap pakai
- Layout yang menarik dan mudah dibaca

## 🏗️ Struktur Modul Ajar

### A. Informasi Umum
- Identitas Modul (Penyusun, Instansi, Mata Pelajaran, dll.)
- Kompetensi Awal
- Dimensi Profil Lulusan (sesuai Kurikulum Berbasis Cinta)
- Capaian Pembelajaran
- Tujuan Pembelajaran
- Model dan Pendekatan Pembelajaran
- Sarana dan Prasarana

### B. Komponen Inti
- **Langkah-langkah Pembelajaran**:
  - Kegiatan Pendahuluan (Berkesadaran, Bermakna)
  - Kegiatan Inti (Konstruksi Pengetahuan, Diskusi Kelompok, Presentasi & Klarifikasi)
  - Kegiatan Penutup
- **Asesmen/Penilaian**:
  - Asesmen Diagnostik
  - Asesmen Formatif
  - Asesmen Sumatif
  - Rubrik Penilaian

### C. Kegiatan Pengayaan dan Remedial
- Program Pengayaan untuk siswa advanced
- Program Remedial bertahap untuk siswa yang membutuhkan bantuan

## 🚀 Cara Penggunaan

1. **Isi Identitas Modul**
   - Masukkan nama penyusun, instansi, dan informasi dasar
   - Pilih mata pelajaran dari dropdown
   - Tentukan fase/kelas, semester, dan tahun pelajaran

2. **Pilih Pendekatan Pembelajaran**
   - Centang pendekatan yang diinginkan (bisa multiple)
   - Sesuaikan dengan karakteristik siswa dan tujuan pembelajaran

3. **Tentukan Model Pembelajaran**
   - Pilih model pembelajaran (PBL, PjBL, Discovery Learning, dll.)

4. **Isi Kompetensi & Tujuan**
   - Masukkan kompetensi awal siswa
   - Tulis capaian dan tujuan pembelajaran

5. **Generate dengan AI**
   - Klik tombol "Generate Modul Ajar dengan AI"
   - Tunggu proses AI (±2 detik)

6. **Download Hasil**
   - Preview hasil di browser
   - Klik "Download Word (.docx)" untuk mendapatkan file
