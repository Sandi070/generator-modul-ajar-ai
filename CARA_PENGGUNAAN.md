# 📖 Cara Penggunaan Generator Modul Ajar AI

## 🚀 Quick Start

### 1. Buka Aplikasi
- Buka file `index.html` di browser (Chrome, Firefox, atau Edge)
- Atau double-click file `index.html`

### 2. Isi Form
Lengkapi semua field yang diperlukan:

#### Identitas Modul
- **Nama Penyusun:** Nama Anda
- **Instansi:** Nama sekolah/madrasah
- **Mata Pelajaran:** Pilih dari dropdown (Kimia, Fisika, Matematika, dll)
- **Fase/Kelas:** Contoh: E/X, F/XI
- **Semester:** Ganjil atau Genap
- **Tahun Pelajaran:** Contoh: 2025/2026
- **Materi/Topik:** Contoh: Pereaksi Pembatas
- **Alokasi Waktu:** Jumlah jam pelajaran (JP)

#### Pendekatan Pembelajaran (Pilih yang sesuai)
- ☑️ Pendidikan Karakter
- ☑️ Moderasi Beragama
- ☑️ Bilingual (Indonesia-Inggris)
- ☑️ Bilingual (Indonesia-Arab)
- ☑️ Berbasis TIK
- ☑️ Pembelajaran Inovatif

#### Model Pembelajaran
Pilih salah satu:
- Problem Based Learning (PBL) - Recommended untuk Kimia
- Project Based Learning (PjBL)
- Discovery Learning
- Inquiry Learning
- Cooperative Learning
- STEAM

#### Kompetensi & Tujuan
- **Kompetensi Awal:** Opsional, bisa dikosongkan (akan di-generate otomatis)
- **Capaian Pembelajaran:** Wajib diisi
- **Tujuan Pembelajaran:** Wajib diisi (pisahkan dengan enter untuk multiple tujuan)

### 3. Generate Modul
- Klik tombol **"🤖 Generate Modul Ajar dengan AI"**
- Tunggu beberapa detik
- Hasil akan muncul di bawah form

### 4. Download Modul
- Klik tombol **"📥 Download Word (.docx)"**
- File akan otomatis terdownload
- Buka file dengan Microsoft Word atau LibreOffice

## 🎯 Fitur Khusus

### 1. AI Generator Cerdas
Untuk topik tertentu (seperti Pereaksi Pembatas), aplikasi akan:
- ✅ Generate soal dengan perhitungan spesifik
- ✅ Menyesuaikan langkah pembelajaran dengan topik
- ✅ Memberikan contoh konkret dan demonstrasi

### 2. Pendekatan Pembelajaran
Memilih pendekatan akan mempengaruhi:
- Dimensi profil pelajar Pancasila
- Sarana dan prasarana yang direkomendasikan
- Aktivitas pembelajaran

### 3. Model Pembelajaran
Setiap model memiliki fase berbeda:
- **PBL:** Orientasi → Organisasi → Penyelidikan → Penyajian → Evaluasi
- **Discovery:** Stimulation → Problem Statement → Data Collection → Verification
- **Cooperative:** Pembentukan Kelompok → Kegiatan Kelompok → Presentasi

## 🔧 Troubleshooting

### Masalah: Form tidak bisa di-submit
**Solusi:** Pastikan semua field yang wajib (required) sudah diisi

### Masalah: Hasil terlalu generik
**Solusi:** 
1. Gunakan topik yang didukung (Pereaksi Pembatas, Stoikiometri, Ikatan Kimia)
2. Atau aktifkan AI API di `config.js` untuk topik lainnya

### Masalah: Download tidak berfungsi
**Solusi:**
1. Cek console browser (F12) untuk error
2. Jika Word gagal, akan otomatis download sebagai HTML
3. HTML bisa dibuka di Word dan di-save as .docx

### Masalah: Konten tidak sesuai
**Solusi:**
1. Edit field "Capaian Pembelajaran" dan "Tujuan Pembelajaran" dengan lebih spesifik
2. Pilih model pembelajaran yang sesuai dengan materi
3. Setelah download, edit manual di Word sesuai kebutuhan
