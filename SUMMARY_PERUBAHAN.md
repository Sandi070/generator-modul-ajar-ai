## 🆘 Troubleshooting

### Masalah Umum:

#### 1. Git not found
```
Solusi: Install Git dari https://git-scm.com
```

#### 2. Permission denied (GitHub)
```
Solusi: Gunakan Personal Access Token
Buat di: https://github.com/settings/tokens
```

#### 3. Deploy gagal
```
Solusi:
1. Cek build logs di dashboard
2. Pastikan semua file ter-commit
3. Cek file .gitignore
```

#### 4. Aplikasi tidak berfungsi
```
Solusi:
1. Buka console browser (F12)
2. Cek error di console
3. Pastikan CDN libraries ter-load
```
