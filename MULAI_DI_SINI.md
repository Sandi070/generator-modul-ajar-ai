# 🎯 MULAI DI SINI - PANDUAN SUPER CEPAT

## ⚡ 3 LANGKAH MUDAH (5 MENIT)

### 1. Dapatkan API Key (GRATIS)
1. Buka: **https://makersuite.google.com/app/apikey**
2. Login dengan Google
3. Klik **"Create API Key"**
4. Copy API key (contoh: `AIzaSyD...`)

### 2. Isi API Key
1. Buka file **`config.js`**
2. Cari baris: `apiKey: '',`
3. Paste API key: `apiKey: 'AIzaSyD...',`
4. Save (Ctrl+S)

### 3. Test Aplikasi
1. Refresh browser (Ctrl+F5)
2. Isi form modul ajar
3. Klik **"Generate dengan AI"**
4. Tunggu 10-30 detik
5. Lihat hasilnya! 🎉

## 🎓 HASIL YANG DIDAPAT

### SEBELUM (Template):
```
1. Jelaskan tentang pereaksi pembatas
2. Sebutkan faktor yang mempengaruhi
3. Apa pentingnya konsep ini?
```

### SESUDAH (Berkualitas):
```
1. Dalam industri pembuatan amonia, 28 gram N₂ direaksikan dengan 
   6 gram H₂ menurut reaksi N₂ + 3H₂ → 2NH₃. Tentukan pereaksi 
   pembatas dan jelaskan dampaknya terhadap efisiensi produksi! 
   (Ar N=14, H=1)

2. Seorang siswa mencampurkan 10 gram serbuk Mg dengan 100 mL HCl 2M. 
   Reaksi: Mg + 2HCl → MgCl₂ + H₂. Hitung: a) Pereaksi pembatas, 
   b) Volume gas H₂ (STP), c) Massa pereaksi tersisa

3. Mengapa konsep pereaksi pembatas sangat penting dalam industri 
   farmasi untuk meminimalkan biaya produksi dan limbah kimia? 
   Berikan contoh konkret!
```
